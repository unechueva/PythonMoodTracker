# Python Mood Tracker

Учебный проект команды:

- Backend: Django 4+
- Приложение: moodtracker
- Основная модель: Entry (дата, настроение 1-5, заметка, категория, пользователь)
- CRUD для Entry
- Отчёты: недельный/месячный с подсчётом среднего, min/max, streak, рекомендаций
- CSV-экспорт

Структура репозитория:
- moodtracker/       Django приложение
- project_name/      настройки проекта
- manage.py          для запуска Django
- requirements.txt   зависимости
- README.md
- .gitignore
