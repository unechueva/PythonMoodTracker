# PythonMoodTracker

PythonMoodTracker — учебный веб-проект на Django для отслеживания эмоционального состояния пользователя.
Проект позволяет добавлять записи о настроении, просматривать их список и формировать отчёты за неделю и месяц.

Проект выполнен в рамках лабораторной работы по дисциплине, связанной с разработкой веб-приложений на Python.

---

## Функциональность

- Регистрация и авторизация пользователей
- Добавление записей о настроении
- Просмотр списка записей
- Формирование отчётов:
  - за неделю
  - за месяц
- Разделение данных по пользователям

---

## Используемые технологии

- Python 3
- Django
- SQLite
- HTML / CSS (шаблоны Django)

---

## Структура проекта

- `moodproject/` — основной Django-проект
- `moodtracker/` — приложение для работы с настроением
- `templates/` — HTML-шаблоны
- `migrations/` — миграции базы данных
- `manage.py` — файл управления Django-проектом

## Установка и запуск проекта

### 1. Клонирование репозитория

git clone https://github.com/unechueva/PythonMoodTracker.git
cd PythonMoodTracker

### 2. Создание и активация виртуального окружения

Windows:
python -m venv venv
venv\Scripts\activate

Linux / macOS:
python3 -m venv venv
source venv/bin/activate

### 3. Установка зависимостей

pip install django

### 4. Применение миграций

python manage.py migrate

### 5. Запуск сервера разработки

python manage.py runserver
После запуска приложение будет доступно по адресу:
http://127.0.0.1:8000/
